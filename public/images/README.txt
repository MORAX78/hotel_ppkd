PANDUAN LOGO PPKD HOTEL
=======================

1. Download logo PPKD Jakarta Pusat dari website resmi atau gunakan logo yang sudah ada.

2. Simpan file logo dengan nama:
   - ppkd-logo.png  (untuk sidebar & print header)
   
   Letakkan di folder ini: public/images/ppkd-logo.png

3. Ukuran yang direkomendasikan:
   - Minimal: 64x64 px
   - Ideal: 128x128 px atau 256x256 px
   - Format: PNG dengan background transparan (terbaik)

4. Sudah ada file ppkd-logo.svg sebagai placeholder sementara.
   Ganti dengan file PNG asli logo PPKD.

CATATAN:
- Sidebar: logo ditampilkan ukuran 32x32px dalam lingkaran
- Print header: logo ditampilkan ukuran 56x56px dalam lingkaran emas
- Jika file gambar tidak ditemukan, sistem akan menampilkan ikon hotel sebagai fallback otomatis
